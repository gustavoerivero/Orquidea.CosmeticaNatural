Proyecto "Orquídea. Cosmetica Natural" para la empresa de cosméticos y productos de belleza bajo el mismo nombre. Este proyecto consiste en un sistema para manejar la información de insumos y productos terminados como también el flujo de compras, ventas, ingresos y gastos. Este sistema es una aplicación de escritorio que usa el patrón de diseño MVC (Modelo-Vista-Controlador), con Netbeans como IDE, lenguaje de programación Java y PostgreSQL como manejador de Base de Datos.
